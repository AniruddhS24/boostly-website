## Getting Started

### Installation

```bash
# Install dependencies
npm install
```

### Development

```bash
# Run the development server
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser to see the result.

### Build

```bash
# Create production build
npm run build

# Start production server
npm start
```

## Components

### Hero
Large headline with primary and secondary CTAs, featuring a gradient background and placeholder for marketing dashboard visual.

### Problem
Highlights pain points that sports and fitness businesses face with marketing.

### Solution
Introduces the AI marketing manager with key benefits.

### Features
5 feature cards showcasing AI capabilities:
- AI Social Media Manager
- AI Flyer Designer
- SEO Engine
- Website Copy Improvements
- Monthly Growth Insights

### How It Works
3-step process showing how easy it is to get started.

### Pricing
3 pricing tiers (Starter, Pro, Elite) with feature comparisons.

### FAQ
Accordion-style FAQ section with common questions.

### Final CTA
Bold call-to-action section encouraging sign-ups.

### Footer
Complete footer with links, social media icons, and legal information.

## Customization

### Colors
The theme uses custom color palettes defined in `tailwind.config.ts`:
- Primary: Blue tones
- Success: Green tones

### Typography
Uses Inter font family from Google Fonts.

## Tech Stack

- **Framework:** Next.js 14
- **Styling:** TailwindCSS 3.3
- **Language:** TypeScript 5
- **Font:** Inter (Google Fonts)
